# vggnet
Using VGG-13 to classify CIFAR-10 Images

~91.5% Test Accuracy

Number of Epochs: 110

Batch Size: 128

Learning Rate Schedule: Initialize to .01, reduce by a factor of 10 at 80 and 100 epochs

Weight Decay: 5E-4

Original Paper: 
Simonyan, K., & Zisserman, A. (2015). Very deep convolutional networks for large-scale image recognition. 3rd International Conference on Learning Representations (ICLR 2015), 1â€“14.
